Chillispot Help Page 
ChilliSpot-Max-Input-Octets 	
	Maximum number of octets (bytes) the user is allowed to transmit. 
After this limit has been reached the user will be disconnected. 


ChilliSpot-Max-Output-Octets
	Maximum number of octets (bytes) the user is allowed to receive. 
After this limit has been reached the user will be disconnected.


ChilliSpot-Max-Total-Octets 
	Maximum number of octets (bytes) the user is allowed to transfer 
(sum of octets transmitted and received). After this limit has 
been reached the user will be disconnected.
